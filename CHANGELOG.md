\#\# \[1.0.0\] \- 04-14-2024

\#\#\# Added

\*   Initial script (\`Microsoft\_AIsearch.py\`) created.  
\*   Fetches data from Microsoft Learn Catalog API.  
\*   Implements semantic filtering using TF-IDF and Cosine Similarity (\`scikit-learn\`).  
\*   Adds course recommendation based on semantic similarity.  
\*   Includes text summarization using NLTK to create program overviews.  
\*   Formats output into "Program Segments" with core and related courses.  
\*   Implements pagination for displaying results.  
\*   Adds basic error handling for API requests and NLP tasks.  
\*   Includes automatic download prompt for NLTK 'punkt' tokenizer.  
\*   Basic documentation files (\`README.md\`, \`CONTRIBUTING.md\`, \`CHANGELOG.md\`).  
\*   Dependency list (\`requirements.txt\`).  
\*   Instructions for dependency installation and VS code app launch (\`INSTRUCTIONS.md\`).
